      JsonExtensionsToJson(Object, Boolean, Boolean, String) Method

      Serializes any object to a non-indented JSON string masking the specified properties with the null values. The resulting JSON string will exclude null values.

      Definition

      Namespace: DotNetExtras.Security.Json
      Assembly: DotNetExtras.Security (in DotNetExtras.Security.dll) Version: 1.0.8
      C#
      public static string ToJson(
	this Object? source,
	bool indented,
	bool useOriginalCase,
	params string[]? maskedProperties
)

      Parameters

      source  Object
      Source object.
      indented  Boolean
      If true, serialized JSON elements will be indented.
      useOriginalCase  Boolean
      If true, the original property names will be used (unless they are overwritten by the JSON serialization attributes); otherwise, the camelCase notation will be used.
      maskedProperties  String
      List of properties to mask (use full property names for nested properties, e.g. PasswordPolicy.Password).

      Return Value

      String
      JSON string with the masked values of the specified properties.

      Usage Note

      In Visual Basic and C#, you can call this method as an instance method on any object of type Object. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).

      Example

      C#
      User user = new User()
{ 
    Name = "John", 
    Age = 30, 
    Password = "sensitiveValue" 
};

// The password property value will hold null; the JSON string will be indented.
Console.WriteLine(user.ToJson("Password", true, true));
