      Crypto Methods

      Methods

      Decrypt Decrypts cipher text using a 256-bit AES key and a 128-bit initialization vector (IV) derived from the caller-provided password and random 8-character salt.
      Encrypt Encrypts plain text using a 256-bit AES key and a 128-bit initialization vector (IV) derived from the caller-provided password and random 8-character salt.
