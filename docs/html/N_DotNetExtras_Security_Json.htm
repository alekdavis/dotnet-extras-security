      DotNetExtras.Security.Json Namespace

      Provides classes to simplify JSON operations on sensitive data, such as serialization of passwords and other secrets.

      Classes

      CharMaskJsonConverter Masks a string property with a specified character during JSON serialization using `System.Text.Json` (STJ).
      HashMaskJsonConverter Masks a string property with a specified hash value during JSON serialization using `System.Text.Json` (STJ).
      JsonExtensions Implements extension methods for serializing objects to JSON strings with masked values of the specified properties.
      LiteralMaskJsonConverter Applies a literal string mask to a string property during JSON serialization using `System.Text.Json` (STJ).
      MaskAttribute Masks a sensitive string property value during an object-to-JSON-string conversion via the System.Text.Json (STJ) serialization.
